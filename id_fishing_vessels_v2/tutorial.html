Identify Fishing Vessels

Can you identify fishing vessels?

Thank you for helping Global Fishing Watch!

The goal of this project is to identify what type of fishing vessel you see on the map. You will be shown the path of a vessel for one month, along with a link to marine tracker (which may have additional information about the vessel, including images of it and other information). If you think it is a fishing vessel, you can categorize it as one of the following:

  • Longliner
  • Purse Seiner
  • Trawler
  • Reefer/Fish Carrier
if you don't think it is a fishing vessel, obviously select "not a fishing vessel." If you don't' know don't worry and just click "Don't know." And if the data looks bad, there isn't enough, or it looks like more than one vessel is using the mmsi, select one of those.
